소개

자짤 갤러리(dcselfie.win)는 DCInside의 여러 갤러리에 올라온 자짤을 한곳에 모아 핀터레스트처럼 보여주는 실시간 피드입니다.

어떻게 동작하나요

  • 운영자가 지정한 특정 갤러리들에서 자짤을 주기적으로 가져옵니다.
  • 가져온 이미지는 최신순으로 화면에 쌓이고, 약 5초마다 자동으로 갱신됩니다.
  • 각 이미지는 일정 시간(기본 3시간)이 지나면 자동으로 사라지며 영구 저장되지 않습니다.
  • 이미지 제목을 누르면 원본 게시글로 이동합니다.
  • 추가했으면 하는 갤러리가 있으면 갤러리 추가 요청으로 알려주세요.

운영 정책

본 사이트는 비영리/취미 목적으로 운영됩니다. 표시되는 이미지는 외부 커뮤니티의 공개 게시물에서 비롯되며, 권리 침해가 우려되는 콘텐츠는 요청 시 신속히 삭제합니다. 자세한 내용은 개인정보처리방침을 참고하세요.
